Вставка редактора формул

 

С помощью редактора формул можно создавать сложные формулы, выбирая символы с панели инструментов и вводя переменные и числа. При создании формул размер шрифтов, интервалы и форматы автоматически регулируются в соответствии с правилами записи математических выражений. Изменять форматирование можно и в процессе работы.

1.       Укажите место для вставки формулы.

2.       На панели Текст ленты Вставка нажмите кнопку Объект.

3.       Откроется диалоговое окно Вставка объекта.

Диалоговое окно Вставка объекта

4.       В списке Тип объекта выберите Microsoft Equation 3.0.

Выбор Microsoft Equation 3.0

Если редактор формул недоступен, его необходимо установить.

5.       Нажмите кнопку OK.

6.       Создайте формулу, выбирая символы на панели инструментов Формула и вводя переменные и числа с клавиатуры.

Панель инструментов формула

ü  Верхняя строка панели инструментов Формула содержит более 150 математических символов.

 

ü  Нижняя строка используется для выбора разнообразных шаблонов, предназначенных для построения дробей, интегралов, сумм и других сложных выражений.
