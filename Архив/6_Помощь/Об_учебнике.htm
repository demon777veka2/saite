Об электронном учебном пособии "Microsoft Word 2007"

Рецензенты: Пимонов А.Г., профессор, д.т.н., заведующий кафедрой вычислительной техники и информационных технологий

Сарапулова Татьяна Викторовна, Тайлакова Анна Александровна Microsoft Word 2007 : учебное пособие [Электронный ресурс] : для студентов экономических специальностей/ Т. В. Сарапулова, А. А. Тайлакова. – Электрон. дан. – Кемерово : ГУ КузГТУ, 2010. – 1 электрон. опт. диск (CD-ROM); зв.; цв.; 12 см. – Систем. требования : Pentium IV; ОЗУ 256 Мб; Windows XP/Vista/7; (CD-ROM-дисковод); мышь. – Загл. с экрана.

Электронный ресурс предназначен для изучения возможностей использования текстового редактора "Microsoft Word 2007". Учебное пособие включает теоретическую базу, практические задания, выполнение которых закрепит полученные знания, а также тесты для самопроверки. Электронное учебное пособие создано посредством технологии HTML, видеоролики – в программе CamStudio 2.0. Для запуска учебника можно использовать различные интернет-браузеры, такие как Internet Explorer, Opera, Mozilla и др. Учебник предназначен для работы на лабораторных занятиях, так и для самостоятельной работы всех экономических специальностей.
